# 次世代型法人決済口座開設審査モデル

本リポジトリは、法人口座開設審査における  
「決済機能」と「与信判断」の分離可能性について整理した、制度設計上の試案です。

特定の金融機関・個別審査結果を批判することを目的とするものではなく、  
中小企業・新設法人・再起業者等に対する金融包摂の観点から、  
決済専用口座モデルおよび取引実態に基づく段階的信用形成モデルを検討するものです。

## 内容

- 現行審査における課題整理
- 決済専用口座と与信連動口座の分離案
- 事業実態確認に基づく口座開設フロー
- 取引履歴に基づく将来的な与信判断モデル

## 注意

本資料は制度設計上の考察であり、法律・金融実務上の助言ではありません。
